WasmEdge Rust KNative with Plugin Demonstration
===
This repository verifies a KNative setup for usage of Wasm container services that use WasmEdge plugins.
For CI/CD setup and KNative integration refer to [tscng/wasmedge-rust-plugin-demo](https://github.com/tscng/wasmedge-rust-plugin-demo) and [tscng/wasmedge-rust-knative-demo](https://github.com/tscng/wasmedge-rust-knative-demo).
